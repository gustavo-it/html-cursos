Tabelas

Nome Sobrenome Idade
Maria Fernanda 25
Camila Ferreira 32

Para iniciar uma tabela utilizamos a tag table. Em seguida temos que definir uma linha. Linhas são definidas pela tag tr. Dentro de cada linha temos as colunas, que são definidas pela tag td. Opcionalmente, caso queira definir melhor a separação do cabeçalho e o corpo da tabela, você pode utilizar as tags thead e tbody. Para indicar o título de uma tabela, utilizamos a tag th no lugar de td.
