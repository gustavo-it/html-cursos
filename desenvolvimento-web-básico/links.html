Por padrão os links sempre serão abertos na mesma aba/janela. Para abrir em uma nova aba/janela, passamos o atributo target com o valor "_blank"

Google

Se quisermos fazer download de algum arquivo, podemos passar o endereço do arquivo em href e passar para a tag a o atributo “download”.

Outro uso muito poderoso da tag a é a criação de âncoras na página. Isso faz com que quando o usuário estiver em uma página muito grande, ao clicar em um link a página irá rolar até o conteúdo marcado com a âncora. Para isso, basta passar um atributo “id” com um nome para o elemento que se quer localizar, e passar esse nome no href com “#”.
